# Time-Complexity-Extension
A browser extension that gives the time complexity of various random algorithms everytime you open the extension or click on the Next button while using it.

<img src="./imgs/screenshot.png" alt="screenshot" width="100%">

### How to install the unpacked extension in Chrome ?

- Go to **settings** chrome

<img src="./imgs/settings.png" alt="settings chrome" width="100%">

- Click on the button **"extension"**

<img src="./imgs/extension.png" alt="settings chrome" width="100%">

- Activate **developer mode**

<img src="./imgs/developerMode.png" alt="settings chrome" width="100%">

- Click on the button to load **extension** folder

<img src="./imgs/load.png" alt="settings chrome" width="100%">

- Load the folder
